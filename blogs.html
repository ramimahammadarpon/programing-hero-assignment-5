Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select element in the DOM. The ways are given below:

  1. Using document.getElementById("id"): This allows user to select element using Id Attribute.
  2. Using document.getElementsByTagName("tag-name"): This allows user to select element using tag name.
  3. Using document.getElementsByClassName("class-name"): This allows user to select element using class name.
  4. Using document.querySelector("attribute selecor+attribute name"):This allows user to select one element using selector and name. For Example, to select class color we need to write: document.querySelector(".color").
  5. Using document.querySelectorAll("attribute selecor+attribute name)This allows user to select more than one element of same name using selector and name. For Example, to select elements with class colors we need to write: document.querySelectorAll(".colors").

Question-2: What is the difference between innerHTML, innerText, and textContent ?

There are a major difference in textContent, innerHTML and innerText is:

  1. TextContent: This returns all the contents in an element regardless of its visibility.
  2. innerText: This returns only visible text content within an element.
  3. innerHTML This returns elements inside tags.
